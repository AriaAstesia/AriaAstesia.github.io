四季刻歌


何時か見た影法師
遠い遠い日の想い出は
春が過ぎ夏が来る
刻は止まることなく


分かってた、はずだったのに
どうしても記憶から消せずに
あの日から止まったままの
私の中の時計


いつか、夢が、終わるのならば
すべて、すべて、無駄だというの？
時は、進み、戻りはしない
きっと、永久に、孤独なのでしょう


君の世界の時は
私には少し早すぎて
秋も過ぎ冬が来て
季節は移ろいでく


掴んでた筈だったのに
また一つ一つと消えていく
いつか来る、別れのことは
分かりきってた事で


あの日、見せた、君の笑顔は
今日は、今日は、どこにもないの
当たり前の幸せでさえ
私の前で
消えてしまうの？


「僕は、君を、一人にしない」
あの日、ここで約束したの
もしも、時を止められたなら
決して二度と、進まないでと


世界、刻に、切り離された
一人、私、まだ出来ること
何度、季節、変わるとしても
枯れる日まで歌い続けて


君に、君に、届くのならば
明日に、明日に、響いておくれ
「ずっと、君の、そばにいたい」と
風に乗せて
遥か彼方へ…




今天下午英语课听 Randy 说 777 可能要退役了，有感而发。


可惜在写博客的时候草稿不见了，于是只好重写一遍。


很棒的一首歌，在最后的时候我甚至想再贴一遍整首歌，不过还是只选了两句出来。


本文中所有人名均用 ID 代替。


很久以前
我第一次接触到英雄联盟是在小学的某个暑假，我的母亲的一个朋友带我去她的亲戚家里玩，我便看到有人玩，于是在旁边围观，自己也想玩。我借过他的号，但因为怕掉分被拒绝了，所以我是偷偷用母亲的号注册的。当时是真的完全不知道这个游戏该怎么玩，只会自己乱搞，出门就是一块蓝 or 红水晶，也没有看过聊天框，不知道自己有没有被喷过。大概率就是了。


由于家里没有合适的电脑，小 A 了一段时间。


初中的时候，父亲搬回来一台配置还不错的台式，第一时间就把英联下了。在暑假里面趁他去上班在家里疯玩，被抓了几次，骂的很惨。但是由于加入了你校睿智的特长班开始了留校生活，便又 A 了。


这便是大概的起始了，一个简单的小学生的英雄联盟。没有排位，没有技术，没有比赛，只有一个傻逼自己乱玩，甚至连自主训练都不知道。


重新开始
我是在高一开学前的暑假回学校上竞赛时再次见到这个游戏的。当时是看到 YZ 在玩，就坐在他旁边看，也一起跟着看了几场比赛，现在只记得 RNG vs SKT 那场了。


刚刚开始只是为了和大家找个话题，顺便打发时间，也不明白玩法，什么意识和技巧完全不知道，只是听了几句解说，算是装成一个云玩家吧。


所以我真正开始了解这个游戏算是在高一吧。开始停课后机房也是换了几次游戏，像是什么 MC 和 Snake，最后还是玩起了 LOL。一开始我是窝在一旁快乐以撒的，但是作为一个单机游戏，最大的缺点就是没法联机，没法在游戏里和朋友一起努力撕逼。算是在 NOIp 前最后几天我才和大家玩了几把，之前都是在围观。


印象最深的一局是有天周五杨爷爷跑过来和我们玩，还有 YZ，Tranced，namo 四个人开黑。中途 zhuml 突然进来查房，导致四人直接挂机。不过最后还是很嗨地赢了，当时我和 HPL 就在一旁叫。


不过可惜，很多人都走了，当时高二只有 YZ 和 HPL 留了下来。


前辈们
和大家在一起上课是初三的事，当时刚刚考完 NOIp 普及组，被调过去了。不过最开始的时候连大家的名字都不知道，也不怎么说话，过了将近一年，才在高一前坐了过去和大家开始聊天，慢慢熟络起来。


我当时坐在 YZ 的旁边，另一边是 Yonda，和我一起过去了。可以说这两个人是陪我最久的人了吧，如果我是女生，说不定我就和他们表白了。


当时中午还是不用回寝室的，每天 YZ 就在努力上分，一天可以来个两三把。我就在旁边看着，听他给我普及一些操作，意识，小技巧，也和他一起喷菜鸡队友，不过都是他说这个人该怎样怎样打，我说对啊这人傻逼吧。


namo
在国庆的时候我们去了雅礼集训，算是一种典型的“集训”吧，挺好笑的，就是神题和天书。所以在后面我们就不去停课了，每天下午就在宾馆里休闲娱乐。我在和 Yonda 一起玩风暴英雄，最后一天和 namo（一个长的很好看的男孩子。如果我是女生我绝对要表白！虽然他有时候有点浪？见到一个小姐姐就说漂亮，但我还是无法理解为什么他的前女友和他分手。随口说说又不怎么，人家那么好的人，是你想要就要想丢就丢的吗？）联机玩了一晚上无主之地 2。最后还发生了一些并不愉快的事。


回来之后我们就在小机房（也就是现在的化学实验室。这是我很失落的原因之一，不过主要还是我太菜。）开始停课，我和大家也聊的开了。我旁边坐的是 namo，他也是 LOL 的爱好者。


说来也巧，好像每年 NOIp 停课的时候总是有 S 赛。所以 S7 开始后大家都开始看比赛。我也就跟着看了一些，不过主要还是在自己玩自己的。毕竟这个我真的不了解，也不太懂解说在说什么，只能看个热闹。


他们当时也经常在群里聊比赛的事。由于 OI 浓厚的膜人，namo 有时候还喜欢开匿名在那里膜人，有时候还会自己和自己演戏，于是我们戏称他“戏精”。


之后是决赛，不过 LPL 的队伍已经被淘汰完了，鸟巢大战的是 SKT 和 SSG。恭喜安掌门。


临近联赛，杨爷爷有时候周末也会过来和我们聊一下，本来是过来给我们相当于是辅导的，不过最后还是一起娱乐了起来，也发生了之前说的那件事。很高兴。


可惜的是并不是所有人都会一直陪在我身边，大家除了竞赛还有别的自己要做的事，于是在 NOIp 之后，有的人满意离开，也有人失意退出。和很多人最后一次见面，也就是在之后获奖照相了。


之后有一场考试，怎么说呢，可以说是一次笑话也不为过。我也记不得具体是啥了，虽然说得好听，但总之完全没有用就是了，退役的也就只有 namo 要来。还是很伤心的，看到他忘记线段树怎么写，在考场上睡觉，最后暴零，总有一种说不出的失落。在回来的车上他和我聊了一小下人生，我告诉他我想当老师，他告诉我：




我无法理解的就是，为什么我们教育下一代的人，不是上一代中最优秀的人。




当时我差点哭出来。离别真的很难受。


很多人都只是别人生命中的过客，走了，就不会再回来了。


为什么，不是所有人都能有一个好的结局呢？


安得广厦千万间？


namo 便再也没有出现过。也许他是被禁止用 qq 了吧，总之我很久没见过他了。本来要是他在群里说话，大家都会极有默契地选择匿名然后开始疯狂发言（演戏）。


唯一一次见到他是我和她出去吃饭，偶遇他和同学一起走下来，脸上还是那样，一抹轻佻的笑容。见到我有女朋友，一种调侃的语气：




哟，TG。




之后我对她说，那是一位我很尊敬的前辈。


YZ
和妹狗狗一起很好玩，因为我和他重叠的爱好还蛮多的。


妹狗狗 NOIp 考了 430 左右，并没有去成任何 wc。


当时也发生了一些极其不愉快的事，可以说是我开始讨厌竞赛的直接原因。


总之快乐（并不）的寒假生活就开始了。


也没有什么事吧，总之就那样到了省选，具体可以看SCOI2018游记。

本来省选完了妹狗狗是说要提前一年高考一本去川大，在家里自学，但是好像川大又怂了就没收妹狗狗，于是就回学校了。


之后也是又一场考试，是 APIO2018，我算是跟去混饭吃吧。不得不说伙食还是很好的，比你校垃圾猪饲料不知道高到哪里去了。


可惜的是妹狗狗去是因为报名交了钱没法退，是什么牌我也记不得了。


总之我也没有再见过他，连偶遇都没有，只有在水群的时候还能说说话。


Tgotp
在北京参加 APIO 的时候 Yonda 和 YZ 分在了一间房里，我是和一个四中的人分在一起的，然后他们就过来找我换房间，我就问他另一个人是谁，他给我说是一个叫 Tgotp 的。


巧了。


这人我认识。


怎么会呢？明明向我这样不愿意与人交往的人，应该不会认识其他学校的人才对的啊。


我第一次见到 Tgotp 是在初三暑假和高一一起学竞赛的时候。印象深刻是 zhuml 之后数次提到了他。


第一天来写了个替罪羊树，NOIp 后听说去北京了很长时间，结果省选炸了，于是又留了一级。他本人也在 qq 上找过我几次。


我进去的时候他也吃了一惊，他正在和朋友一起 ow，然后 YZ 就跑过来约他一起 LOL，我找他借了小号一起开黑。


挺疯的吧，每天到凌晨一点，第一天下午就开始窝在宾馆，之后就只去参加了比赛。


祝愿他今年能有个好结局。


同学们
在省选爆炸后，什么事都没有了，于是被叫回去和同年级的一起上课。


也没什么好留恋的了吧。高二就只剩 HPL 一个人，我和他关系没有那么好，算是比较好的同学关系而已。


刚开始回来的时候并不愉快，因为有一群不知道是几年级的小朋友有点吵，我当时只记住了 zjk 一个人，因为真的把我恶心到了。


算了，那些低年级的和我也没什么关系了。我是要走的人，他们的未来还长。


总之刚刚一起上课的时候，我是和 Yonda 坐在一个偏僻的角落的。


Yonda
我之前就说过，他是陪我最久的人，可惜他已经走了。


怎么说呢，你校 OI 搞得问题蛮大的，最大的问题就是学生心态普遍有问题。


所以在经过了一年之后，Yonda 和我都是半 AFO 的选手，都开始考虑退役。


这一年，经历的有点多。


说到底，还是你校的底蕴不够。其他学校老师可能也就那样，但是他们有前辈。


我们连一个 Au 都没有。


所以我们的题目质量，训练方式都很成问题。


而且差就差吧，有一些睿智还不愿意承认自己的差。不多评价。牢骚在隔壁NOIP2018游记已经发得差不多了。


反正 Yonda 在省选后是觉得真的搞不下去了，于是决定今年考完联赛就退役。还是发生过一些冲突，但是那是他的私事，我也没必要过问。


所以今年的停课他并没有参加，甚至在停课前把自己 GitHub 上东西全部删了，连号都删了。为了这事我还专门去把 Burnside 引理[1] 的证明补了，算是一种纪念吧。


他说的是以后就算要用到博客也没有对代码的需求了，要是真的还要用博客的话会去 Google Page。


祝您高考顺利。


Randy
Yonda 是一个小小的插曲，和他的认识纯粹是因为这条路太窄，要想走必须贴在一起，所以才认识了，不然我们是不会有交集的吧。


Randy 就不一样了，他是我的初中同学，还是被我拉入竞赛的坑的。本来说的是我来带他，可惜大家都咕咕咕了。


初中的时候，他和我就是班上比较跳的两个，经常做一些哗众取宠之类的事，也被老师批评过几次。每学期第一节英语课就会极有默契地坐成同桌，等着老师来拆 hhhh。


本来的话我以为他也和我一样讨厌学习，不过之后才了解到他思想还蛮正的。在初三后期在认真地学习，高一之后也学得挺认真的。


也许大家都只是讨厌应试？


总之在高二前他竞赛算是在班上排名靠后的，也经常会没有自信，觉得自己是不是不适合学竞赛。不过他学的确实很努力，而且在自学了一短时间之后成为了数一数二的。事实证明只是某些人教的差。


还是蛮可惜的吧，今年发挥不行，一等奖边缘。退役是一定的了。


他也是一位 LOL 玩家，玩的挺好的。S8 我就是和他一起看的，当时因为一些原因我没法正常地休闲，只能跑去别人那里看他们休闲。


在 IG 拿冠军的时候，我们在一起喊 IG 牛逼；在 RNG 被 G2 暴打的时候，我们也会一起骂四保一。


联赛之前他和我开了几把黑，算是最后的纪念。


不过他的主要娱乐方式，或者说机房很多人的主要娱乐方式，都是一个叫 slay 的游戏，一个横版对战 fps。


slay 大概是从洛谷里开始火的吧，感觉是洛谷全民网游，很多 OIer 都在玩。


由于最开始的时候他们是不能带笔记本过来的，所以只能选取相对亲民的方式。


可惜大家还是都走了，那些 ID 不知道还能不能再亮起来。


毕竟会在家里玩 slay 的应该是没有的吧。


汤姆
Tom 也是我的初中同学，这是他的英文名。


一开始我是不打算写他的，因为关系没有那么好。


决定要写是因为他实在是太惨了。去年因为 t1 自己作死写了个高精度在一等奖边缘，然后由于 €€￡ 特派员迟交代码，导致了分数线上调，他正好被卡掉。今年是 Dev 的垃圾编译蒙蔽了他的双眼，没有引用 cstring 就用 memset，然后 3 道 CE。这个问题我以前是遇到过的，当时还在怀疑是不是编译器有问题，后来才知道是 Dev 那个版本编译会自动补全头文件。不过这其实也该算是他自己的问题，因为如果用 NOI Linux 编译的话是不可能过得了的。他说是他不会用，那么在考试之前不学就是他自己的问题了。本来也强调过很多遍关于 Windows 和 Linux 的差别，而且这个就算用 cmd 编译也是过不了的。


算是很不幸运的一个人了吧，至少在竞赛方面。


不知道他能不能学好语文。
